# Life Advice App
Get some real good life advice 

Made for [Curious Byte](https://www.youtube.com/@CuriousByte) but is ideal for anybody wanting a good starter project with all the basics.

### Getting started
npm
```
npm install
npm run dev
```
yarn
```
yarn install
yarn dev
```
pnpm
```
pnpm install
pnpm dev
```
